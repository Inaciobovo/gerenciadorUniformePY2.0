👕 BovoTech - Gerenciador de Uniformes e EPIs
Solução digital para controle de estoque e comprovação de entrega de Equipamentos de Proteção Individual.

📌 Sobre o Projeto
Este sistema nasceu de uma necessidade real observada no setor de Recursos Humanos e Segurança do Trabalho.

O Problema: Em situações de auditoria ou acidentes de trabalho, a localização manual de fichas de entrega de EPIs em arquivos físicos pode levar horas, atrasando tomadas de decisão críticas e expondo a empresa a riscos jurídicos.

A Solução: Uma plataforma web centralizada que digitaliza o controle de entrada e saída de uniformes e equipamentos, permitindo a consulta instantânea do histórico de cada colaborador.

🚀 Funcionalidades Principais
Dashboard Operacional: Visualização rápida do total de itens em estoque, entradas e saídas recentes.

Gestão de Inventário: Cadastro detalhado de uniformes (descrição, valor unitário e quantidade).

Controle de Colaboradores: Registro de funcionários para vinculação de entregas.

Histórico de Movimentações: Registro de quem recebeu o quê e quando (em desenvolvimento).

🏗️ Arquitetura do Sistema (MVC Adaptado)
Diferente da versão inicial em Streamlit, esta versão foca em escalabilidade:

Backend: Rotas em Python controlando o fluxo de dados.

Persistência: Uso de comandos SQL para garantir a integridade dos dados.

Frontend: Interface limpa focada na experiência do usuário (UX).
